# Ground SDK

This fork contains native and Android support for the following Parrot drones:

* Mambo w/Minicam (Wi-Fi only)
* Bebop V1
* Bebop V2
* Disco
* Anafi 4K
* Anafi 4K Thermal

And controllers:

* SkyController V1 (both variants)
* SkyController V2
* SkyController V3
